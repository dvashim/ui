# @dvashim/ui-themes

## 2.0.0

### Minor Changes

- [#1](https://github.com/dvashim/ui/pull/1) [`8e46095`](https://github.com/dvashim/ui/commit/8e46095d21f026d42e2c0ca38730f5cf21a2ceb7) Thanks [@aleksei-reznichenko](https://github.com/aleksei-reznichenko)! - Initial scaffold of the dvashim design system.

  - **Core:** Add StyleX design token primitives — color palette (surfaces, intents, borders, overlay, link, focus), 6-slot intent contract, font size / line-height / weight scales.
  - **Themes:** Add 6 color palettes (light, dark, mono light/dark/dim, hacker) and 4 font density themes (comfortable, compact, large, extra-large).
  - **Components:** Add `Button` (4 variants × 6 intents) and `InteractionOverlay`.
